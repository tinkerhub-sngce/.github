# ⚙️ Built By Us Initiative

> “This is for everyone.”
> — Tim Berners-Lee

## 🧠 About

**Built By Us** is an initiative by TinkerHub SNGCE focused on creating tools for our own community.

Instead of relying on fragmented platforms, we design and build systems tailored to real needs — simple, efficient, and open.

---

## 🚀 What We Do

* Build internal tools for organizations
* Solve real operational problems
* Promote open-source collaboration
* Encourage system-level thinking

---

## 🧩 Current Projects

### Membership & Finance Management System

A lightweight system designed to streamline organizational workflows.

#### ✨ Features

* Member management with role-based access
* Participation and activity tracking
* Finance management (expenses, budgets, records)
* Contribution-based ranking
* Event-wise financial insights
